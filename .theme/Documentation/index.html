A) HTML and CSS structure - top

If you would like to add some space on the page you can simply add a div with a class of space[1-30]. As you may have guessed already: the number is the height of the space in pixels.
Example: 

<div class="space10"></div>

If you would like to add some additional content you could for example: Add a div with a class of space10 and under it you could simply add content such as text and images

If you would like to edit the default font, then open main.css go to body (on line 12), then in the line under you will see the following: 
font-family: 'HelveticaNeue', 'Helvetica Neue', Arial, sans-serif;
Just replace 'HelveticaNeue' with the font you would like to use

B) Edit background color - top

You can choose between 10 different colors: gray, blue, lightBlue, yellow, purple, green, blueGreen, brown, red and pink.
You can apply one of the above colors to the background.

Change the background color
On line 22 in index.html there is the body element, you can change the class of gray to any of the colors above.

C) Edit background pattern - top

You can choose between 8 different patterns:

  • pattern-1
  • pattern-2
  • pattern-3
  • pattern-4
  • pattern-5
  • pattern-6
  • pattern-7
  • pattern-8

Change the background pattern
On line 22 in index.html there is the body element, you can change the class of pattern-1 to any of the patterns above.

If you do not want a background pattern, just remove the pattern class completely

D) Add images to the gallery - top

If you would like to add images to the gallery, then open config.js under resources -> js on line 1

It's just a regular Javascript array, if you're not familiare with arrays then check out: http://www.w3schools.com/JS/js_obj_array.asp

E) Change portrait - top

Replace portrait.png inside resources -> img with your portait.

F) Apply your name accross the template - top

Replace John Doe with you name on the following lines in index.html

  • 45: <h1><span>John</span> Doe</h1>
  • 55: <h1><span>John</span> Doe</h1>
  • 240: <h2>John Doe</h2>
Keep in mind that your first name (in the first two instances) should be wrapped in span tags.

G) Edit the recipient of the email form - top

To edit the recipient of the email form, open contact_handler.php (located under resources -> php), go to line 13 and switch the current email out with yours. 

$receiver = "YOUR EMAIL ADDRESS";

H) Edit the user to get the latest tweet from - top

To edit the user to get the latest tweet from, open config.js, go to line 3 and change the current username out with yours. 

var twitterUsername = 'YOUR USERNAME';

I) Edit social networks - top

To add social networks, open index.html, go to line 171 and change the various usernames out with yours.
To remove social network, just remove the whole li wrapper element.

If you would like to use some other social networks than the displayed ones, you got some other choices. You can see them in the social folder inside the resources -> img folder

I) Skill bars - top

You can add skill bars to the content with a div element, that looks like this: 

<div class="skill">
	<div class="skillBar">
		<div class="skillBarFraction">
			5/10
		</div>
		<div class="skillBarProcentage" style="width:50%;">
			<div></div>
		</div>
	</div>
	<div class="skillName">
		Adobe Illustrator
	</div>
</div>
Remember to edit your fraction accordingly to the procentage.
